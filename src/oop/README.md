# 09 — OOP (Object-Oriented Programming)

**Level:** 🟠 Intermediate  
**Prerequisites:** 01_Basics through 05_Collections  
**Estimated Study Time:** 5–7 hours

## Learning Objectives
- Write classes with encapsulation (private fields + getters/setters)
- Use inheritance and method overriding
- Understand polymorphism through parent references
- Sort objects using Comparator

## Files

| # | File | Concepts Covered |
|---|------|-----------------|
| 1 | `AssociateEncapsulation.java` | Class, private fields, getters, setters |
| 2 | `EmployeeInheritance.java` | Inheritance, @Override, Comparator sort |

## OOP Pillars Summary

| Concept | Keyword | Key Idea |
|---------|---------|----------|
| Encapsulation | `private` + getters/setters | Hide data, control access |
| Inheritance | `extends` | IS-A relationship |
| Polymorphism | `@Override` | Same method, different behaviour |
| Abstraction | `abstract` / `interface` | Hide implementation |
