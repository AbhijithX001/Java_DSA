# 13 — Mini Projects

**Level:** 🔴 Advanced  
**Prerequisites:** 09_OOP + 05_Collections  
**Estimated Study Time:** 2–4 hours each

## What Goes Here
Small console applications that combine multiple concepts.  
These are NOT exam questions — they are for building confidence with real code.

## Planned Projects
- **Library Management** — Book class, add/remove/search books by title
- **Bank Account** — Account class, deposit/withdraw/balance, transaction history
- **Student Grade Calculator** — Student class, average, grade letter, rank
- **Employee Directory** — Add/remove/search employees, sort by department/salary

## How to Add a Project
1. Create a subfolder: `13_MiniProjects/LibraryManagement/`
2. Add all classes inside that subfolder
3. Main class should be named after the project: `LibraryManagement.java`
