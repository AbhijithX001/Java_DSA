# 01 — Basics

**Level:** 🟢 Beginner  
**Prerequisites:** None  
**Estimated Study Time:** 2–3 hours

## Learning Objectives
- Understand basic Java program structure (main method, Scanner, print)
- Practice loops, conditionals, and arithmetic
- Write simple mathematical problems in Java

## Files (study in this order)

| # | File | Topic |
|---|------|-------|
| 1 | `SumOneToN.java` | Formula: n*(n+1)/2 |
| 2 | `MultiplicationTable.java` | for loop |
| 3 | `FibonacciSeries.java` | Two-variable loop |
| 4 | `ReverseNumber.java` | Modulo + division |
| 5 | `PerfectSquare.java` | Math.sqrt() |
| 6 | `PerfectNumber.java` | Divisors + sum |
| 7 | `ArmstrongNumber.java` | Digit extraction + power |

## TCS IPA Relevance
Armstrong, Fibonacci, and Perfect Number have appeared as warm-up questions.
