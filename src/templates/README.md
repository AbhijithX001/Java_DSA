# 12 — General Templates

**These templates are reusable starting points for any Java program.**  
They are not TCS IPA specific — use them for any problem.

| File | When to Use |
|------|-------------|
| `ScannerTemplate.java` | Any program needing user input |
| `ArrayTemplate.java` | Array problems — includes sort, max, second largest |
| `HashMapTemplate.java` | Frequency counting, key-value lookup |
| `ComparatorTemplate.java` | Custom sorting of objects or arrays |
| `OOPTemplate.java` | Any class-based problem — rename and add fields |

## How to Use
1. Copy the relevant template
2. Rename the class to match your problem
3. Fill in the logic
4. Never modify the template file itself
